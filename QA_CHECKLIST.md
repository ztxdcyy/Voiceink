# VoiceInk Runtime QA Checklist

> 目标：在真实 macOS 桌面环境验证 Fn 语音输入全链路。

## 0. 预备条件

- macOS 14+
- 可用 DashScope API Key
- 网络可访问 `wss://dashscope.aliyuncs.com`
- 已执行 `make build` 或 `make run`

## 1. 权限与启动

- [ ] 运行应用后，菜单栏出现 `VoiceInk` 麦克风图标
- [ ] 首次启动可看到辅助功能权限引导
- [ ] 在系统设置中授予 Accessibility 权限
- [ ] 首次录音可触发麦克风权限弹窗并授权

## 2. Settings

- [ ] 打开 `Settings...`
- [ ] 输入并保存 API Key
- [ ] Model 默认值正确（可编辑）
- [ ] `Test` 按钮可返回连接成功

## 3. Fn 键录音与悬浮窗

- [ ] 按住 Fn 出现胶囊悬浮窗
- [ ] 波形随声音大小明显变化（大声更高，安静更低）
- [ ] 菜单栏图标录音态正确切换
- [ ] 松开 Fn 后切到 waiting 状态（spinner）

## 4. 转录与注入

- [ ] 转录增量文本实时更新
- [ ] 收到完成事件后自动注入当前输入框
- [ ] 注入内容与语音一致（中英混杂场景）
- [ ] 原剪贴板内容被正确恢复

## 5. 输入法处理

- [ ] 当前为中文输入法时，注入仍成功
- [ ] 注入后输入法恢复到原状态

## 6. 边界行为

- [ ] 误触（Fn < 300ms）不会触发注入
- [ ] 断网时显示错误并可恢复到 idle
- [ ] 超时（15s）显示“请求超时”并可继续下次会话

## 7. 跨应用验证

至少在以下应用验证一次注入：

- [ ] Notes
- [ ] Safari 地址栏/输入框
- [ ] VSCode 编辑器
- [ ] WeChat/Slack 聊天输入框（如已安装）

## 8. 已知限制（记录）

- Fn/Globe 抑制在不同 macOS 版本可能存在差异。
- 如出现 Emoji 面板弹出，建议将系统 `Globe key` 设为 `Do Nothing`。
